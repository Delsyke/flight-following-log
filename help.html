How to Use

This is a simple program to help us generate the flight following log routes. Steps to use

Overview

  1. Edit and save the file named routings. See below for details on how to edit this file
  2. Open console launcher by double clicking
  3. Type python app.py on the console screen and hit Enter
  4. You are successful if you see Log generated on the console without any other information
  5. Check on this directory and find the excel file named by the date generated
  6. Open the file and copy the information as desired

Editing Routings

For C208 routes start a new line with the syntax shown in the examples below.

04:15 HKNW HKAM HKFH HKNW

07:20 HKNW HKNL HKAK HKLD HKNW

Use the applicable ICAO code whenever available as shown above, or the name of the airstrip where the ICAO code is not available.

05:00 HKNW KALAMA OLARE HKJK

For Oldonyo Wuas, use OLDO

11:00 HKNW OLDO KIMANA HKJK

Multiple aircraft: use an empty line to separate the routings of each aircraft as shown

04:15 HKNW HKAM OLDO HKNW
07:20 HKNW HKLD HKSE HKNW
11:15 HKNW HKMS HKOK HKNW

05:00 HKNW HKAK HKLD HKMF HKMS HKKE HKNW
10:10 HKNW HTKJ HKNW

07:00 HKNW HKOD HKOY HKCC HKNW

Daystop: Enter the route upto the day stop point. Start the afternoon routing on a new line

07:00 HKNW HKGE HKSJ HKMJ HKTB
12:15 HKTB HKOY HKKE HKNW

For DHC8 routes, with exception of the bush flights, use the following syntax

043 HKNW HKMO HTZA HKMO HKNW
051 HKNW HKLU HKML HKNW

081 HKNW HKKI HKNW
053 HKNW HKUK HKNW
047 HKNW HTZA HKMO HKNW

For DHC8 aircraft in bush flights, either fill the log manually or follow the C208 syntax and edit the timings

Errors

Occasionally, you will get errors displayed on the console. Some of the error causes maybe:

  • Incorrect entry of an airfield. For example HMO instead of HKMO
  • Incorrect formatting of a route. For example 0500 HKNW HKSE HKKE HKNW. This will be interpreted like a DHC8 route with an AU code of 050
  • Formatting a DHC8 route with the syntax of a C208 route. For example 06:45 HKNW HKUK HKNW
  • Formatting a C208 route with the syntax of a DHC8 route. For example 023 HKNW HKSB HKNW
  • A new airfield that is still not recognized by the program. In this case, inform Wasike for actioning

In any case, the program will not execute the route with an error. However the other routes will still be executed and you will see the Log generated message. You will also see info on the error and advise that you can correct or complete that section manually. See examples below

Tracking the DHC8 on Long Routes

To prove that we are tracking flights after every 15 minutes, we need to insert additional rows for checks after every 15 minutes interval. We start the first check 15 mins after ATD, and then after 15 minutes until the aircraft lands at the destination.

The number of rows to be inserted depends on the length of the flight, and therefore, the number of checks required. But the following information will be useful. Ref table and examples below

Leg Flight Time # of Checks Rows to Insert
WIL-KIS 45 mins 2 1
WIL-UKA 1hr 10 mins 4 3
WIL-MBA 1hr 10 mins 4 3
WIL-MYD 1hr 10 mins 4 3
WIL-LAU 1hr 20 mins 5 4
LAU-MYD 30 mins 1 0
WIL-ZNZ 1hr 40 mins 6 5
MBA-ZNZ 45 mins 2 1
